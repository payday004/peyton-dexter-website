Tower of Hanoi

Java program that visualizes the famous math and logic puzzle, Tower of Hanoi

Link to GitHub Repo

Towers of Hanoi was my favorite project from CS 2114. The Tower of Hanoi puzzle was invented in the 19th century by French mathematician Ã‰douard Lucas. The puzzle starts with 3 rods and a given number of circular disks of different sizes organized in ascending order on the left most rod. The objective of the puzzle is to move the stack to the right most rod, preserving the order, and without ever placing a larger disk on a smaller disk. My visualization implements the Stack data structure and recursion to show the solution.

Below is a video by Reducible that dose a great job of explaining the solution in depth.

To run the visualization program on your computer, make sure you have Java installed and added to your PATH. download the jar file listed below.

towershanoi.jar

Open Command Prompt and navigate to the directory where the .jar file is saved. In the terminal run the prompt below.

java -jar towershanoi.jar 6

6 indicates that the program will start with 6 disks. You can play around with this number.
